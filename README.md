# Анализ влияния праздников на выручку магазинов
В данном кейсы были найдены ответы на следующие вопросы:
- Какова средняя недельная выручка каждого магазина в праздничные недели?
- Какова средняя недельная выручка тех же магазинов в обычные недели?
- На сколько процентов праздник повышает выручку по сравнению с обычными неделями для каждого магазина?
- Какие магазины сильнее всего реагируют на праздники и где имеет смысл усиливать маркетинговую активность?
# Данные
[Источник данных](https://www.kaggle.com/datasets/manjeetsingh/retaildataset?resource=download)

Используемые таблицы:

- **`sales`** — недельные продажи по магазинам и отделам:
  - `Store` — идентификатор магазина;
  - `Date` — неделя;
  - `Weekly_Sales` — выручка за неделю.
- **`features`** — дополнительные признаки по магазину и неделе:
  - `Store`,
  - `Date`;
  - `IsHoliday` — признак праздничной недели (`'TRUE'` / `'FALSE'`);

## Методология

1. Соединил таблицы `sales` и `features` по `Store` и `Date`, чтобы каждая строка содержала и выручку, и признак праздника.
2. С помощью `CASE WHEN` внутри `AVG` посчитал две метрики по каждому магазину:
   - средняя выручка в праздничные недели;
   - средняя выручка в обычные недели.
3. Рассчитал относительный прирост продаж в процентах.
4. Округлил значения до двух знаков с помощью `ROUND`, вывел результат по каждому магазину и отсортировал по убыванию эффекта праздников.
5. На основе итоговой таблицы построил визуализацию.

## Результаты

- Для большинства магазинов праздничные недели дают заметный рост продаж по сравнению с обычными — вплоть до 18%.
- ТОП 5 лидеров по приросту - это магазины под номерами 35,7,12,15,29.
- Часть магазинов показывает минимальный прирост, что может говорить о слабой реализации промо или особенностях локации.
- Худшие показатели по приросту наблюдаются в магазинах под номерами 36,44,17,38,30.
- Этот анализ помогает приоритизировать маркетинговый бюджет: магазины с высоким процентом прироста логично поддерживать дополнительными акциями и рекламой, а для «слабых» точек искать причины и тестировать другие гипотезы (ассортимент, цены, выкладка).
